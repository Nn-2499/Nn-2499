# 🚀 Masaangna - Interactive Web Development Environment

![Masaangna Preview](https://via.placeholder.com/800x400/050505/00ff88?text=Masaangna+Interactive+Web+Dev+Env)

An interactive web development environment with real-time code editing, live preview, and 5 beautiful color themes. Built with modern web technologies for developers, designers, and creative coders.

## ✨ Features

### 🎨 **5 Beautiful Color Themes**
- **Pink-Red** (#ff1493, #ff4500) - Vibrant and energetic
- **Purple-Blue** (#9400d3, #0000ff) - Deep and mysterious  
- **Cyan-Yellow** (#00ffff, #ffff00) - Bright and futuristic
- **Orange-Green** (#ff8c00, #00ff00) - Warm and natural
- **Rainbow** (#ff00ff, #00ffff) - Vibrant and dynamic

### 💻 **Real-Time Development Environment**
- **Monaco Editor** (VS Code editor) embedded for HTML/CSS/JS editing
- **Live Preview** with instant updates as you code
- **Tab switching** between HTML, CSS, and JavaScript views
- **Keyboard shortcuts**: Ctrl+Enter to run code
- **Project templates** for quick starts:
  - Particle System (Three.js)
  - Audio Visualizer (Web Audio API)
  - Interactive 3D Model (OrbitControls)
  - Live Data Dashboard (simulated real-time data)

### 🖱️ **Interactive UI Elements**
- **Custom animated cursor** system (dot + outline with hover effects)
- **Three.js shader background** with mouse interaction
- **Magnetic navigation elements** that follow cursor movement
- **3D tilt effects** on project cards with glare
- **Animated marquee** with scrolling text
- **Bottom light effect** that changes color/intensity with scroll
- **Random theme generator** (press 'R' key)
- **Theme selector** with 5 preset color schemes

### 📱 **Responsive Design**
- Works seamlessly on desktop, tablet, and mobile devices
- Collapsible sidebar for mobile views
- Adaptive layouts for different screen sizes

## 🚀 How to Use

### Live Demo
You can see a live demo at: [https://yourusername.github.io](https://yourusername.github.io)  
*(Replace "yourusername" with your actual GitHub username)*

### Local Development
1. Clone or download this repository
2. Open `index.html` in any modern web browser
3. Or use the provided script:
   ```bash
   ./run.sh
   ```

### Usage Instructions
- **Switch Themes**: Click the theme buttons in the bottom-left sidebar
- **Random Theme**: Press 'R' key on your keyboard
- **Edit Code**: Use the Monaco Editor on the left/top panel
- **See Preview**: Watch the live preview on the right/bottom panel
- **Run Code**: Press Ctrl+Enter or click the "Run Code" button
- **Load Templates**: Click on project cards to load starter code
- **Responsive Design**: The layout adapts automatically to your screen size

## 🔧 Customization

### Changing Themes
The theme system uses CSS variables that can be easily modified:
- `--accent`: Primary theme color
- `--accent-2`: Secondary theme color

### Adding New Features
The modular structure makes it easy to add:
- New project templates in the `loadProjectTemplate()` function
- Additional UI elements in the HTML structure
- New effects in the JavaScript sections
